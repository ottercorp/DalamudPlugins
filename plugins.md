# CN Neat Plugins

这些是社区制作的插件，您可以在启用游戏内置插件后使用 /xlplugins 在游戏中安装。

如果您想要参与插件简介的汉化项目，请访问 [Dalamud Plugins CN](https://crwd.in/dalamud-plugins-cn) 。
